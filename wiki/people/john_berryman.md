## Appearances

- [[search_agents_john_doug]] — Guest speaker / co-presenter; co-led live coding workshop on wrapping BM25 with agentic intelligence using OpenAI Agents SDK, with [[doug_turnbull]]
- [[ws_5_context_engineering_and_information_retrieval]] — Referenced as upcoming search agents workshop leader; described as early engineer on GitHub Copilot

## Key Contributions

- Co-authored "Relevant Search," the industry standard for bridging business requirements and search configuration
- Framed the agentic search premise: LLMs have common-sense knowledge about language (synonyms, misspellings, that "dress shoes" is a shoe not a dress) and wrapping them around keyword search can overcome lexical mismatch
- Demonstrated building an agentic search wrapper using the OpenAI Agents SDK over Doug's BM25 search array library on the WANDS dataset

## Notable Quotes

- "What happens when you make a search tool and build an agent that is accessing the search? Rather than that straight pass-through deterministic code, you allow the agent to have access to the search tool, the agent has common sense knowledge because these models have been trained on everything." — [[search_agents_john_doug]]
